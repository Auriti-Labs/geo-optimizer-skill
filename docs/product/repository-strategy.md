GeoReady Pro will not be developed directly inside the geo-optimizer main branch.

The open-source repository will remain focused on the audit engine, CLI, public report schema, documentation, and reusable integration primitives.

The paid web platform will live in a separate private repository named geoready-platform.

Shared logic will be exposed through stable interfaces, JSON report schemas, API contracts, or packages, not by mixing SaaS-specific code into the open-source repo.
